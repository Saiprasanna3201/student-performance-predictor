
# Student Performance Predictor

## 📌 Overview
This project predicts student performance based on study hours, attendance, and previous scores using Machine Learning.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## ⚙️ Features
- Data preprocessing
- Linear Regression model
- Model evaluation using Mean Squared Error
- Custom prediction for new student data

## 🧠 How it Works
The model is trained on sample student data and predicts final scores based on input features.

## ▶️ Run the Project
pip install -r requirements.txt  
python main.py
